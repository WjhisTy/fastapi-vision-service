## 🎯 如何使用（3 种方式）

### 方式 1: Windows 快速启动 ⚡

```bash
# 双击运行
run_t2i.bat    # 文生图服务
# 或
run_vl.bat     # 图片理解服务

# 然后访问
http://localhost:8000/docs
```

### 方式 2: Docker 运行（推荐） 🐳

```bash
# 文生图
docker-compose --profile t2i up

# 图片理解
docker-compose --profile vl up

# 访问
http://localhost:8000/docs
```

### 方式 3: 命令行运行 💻

```bash
# 进入项目目录
cd D:\aaaaa\fastapi-vision-service

# 安装依赖
uv sync --extra dev

# 运行文生图
set SERVICE_MODE=t2i
uv run uvicorn app.main:app --reload

# 或运行图片理解
set SERVICE_MODE=vl
uv run uvicorn app.main:app --reload
```
